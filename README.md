## What this playbook covers

| Section | Description |
| --- | --- |
| Home | High-level introduction to the playbook, background, and navigation |
| SOLID Principles | Practical guidance for maintainable object-oriented design |
| System Design | Patterns and trade-offs for scalable, dependable systems |
| Python Best Practices | Production-focused Python conventions and habits |
| Code Review | Review standards, quality checks, and collaboration practices |
| ADRs | A lightweight approach to architecture decision records |
| Engineering Principles | The core beliefs that shape how I build software |
| About | Career background, technical strengths, and current focus |

## Run locally

```bash
pip install -r requirements.txt
mkdocs serve
```

Then open `http://127.0.0.1:8000` in your browser.
